# Multimodal Emotion Recognition (SER & FER) | AIU Research

**🌍 Research Assistant Project | Alamein International University**

A comprehensive AI framework focused on **Speech Emotion Recognition (SER)** and **Facial Emotion Recognition (FER)** systems. This project explores the power of multimodal fusion for robust emotion classification.

## 🛠️ Key Contributions
- Conducting Research on **Speech Emotion Recognition (SER)** and **Facial Emotion Recognition (FER)** systems.
- Leveraging **deep learning architectures** and **multimodal fusion** for high-accuracy classification.
- Collaborating with **Dr. Mohamed Abdelrahman** on developing the core emotion monitoring framework.
- Exploring **Retrieval-Augmented Generation (RAG) pipelines** with **vector databases** and embedding-based search to enhance large-language-model reasoning (collaborating with **Dr. Islam Elgedawy**).

## 🔬 Technologies Used
- **AI Libraries**: PyTorch / TensorFlow (Deep Learning), OpenCV (for FER).
- **Core AI**: Multimodal Fusion, LLMs, RAG.
- **Data Persistence**: Vector Databases (embedding-based search).
- **Domain**: NLP (Speech), Computer Vision (Facial), Sentiment Analysis.

## 🔬 Current Status
- **Active Research**: Status is "In Preparation" for publication.
- **Focus**: Improving classification robustness through joint analysis of audio and visual cues.
