# Autonomous Golf Cart Navigation | James Madison University

**🌍 Research Intern Project | Harrisonburg, VA, USA**

Developed a ROS 2–based perception and control system for autonomous golf cart navigation. This project focused on real-time object detection and path planning in a dynamic environment.


## 🛠️ Key Contributions


- Implemented **YOLOv8 & SSD** algorithms for robust object detection.

- Utilized the **ZED 2i stereo camera** for depth estimation and obstacle detection.

- Designed modular **ROS 2 publisher–subscriber nodes** for system planning and collision avoidance.

- Achieved high-performance system responsiveness with **sub-100ms latency**.

- Integrated actuator interfaces for precise **sensor calibration, motor control, and safety protocols**.

- Leveraged **Docker containerization** for environment consistency and deployment on Linux.


## 🔬 Technologies Used


- **Frameworks**: ROS 2 (Robot Operating System), Docker.

- **AI Algorithms**: YOLOv8, SSD (Single Shot Multibox Detector).

- **Hardward Integration**: ZED SDK, ZED 2i Camera.

- **Tools**: Linux, Bash scripting.

- **Domain**: Robotics, Computer Vision, Autonomous Systems.
