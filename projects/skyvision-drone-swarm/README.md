# SkyVision | Real-Time Sensor Fusion Pipeline for Drone Swarms

**🎓 Graduation Project (Ongoing) | Collaboration With James Madison University**

An AI-powered aerial monitoring and vision system designed for real-time environmental and infrastructure analysis using **drone swarms**. This project is a collaboration with James Madison University (JMU) and Alamein International University (AIU).

## 🛠️ Key Contributions


- Developing an **AI-powered aerial monitoring** and vision system.
- Designing **ROS 2-based multi-drone control** and **computer vision pipelines**.
- Implementing **collision avoidance** with modular **perception and planning nodes**.
- Integrating **LLM-assisted situational analysis** for automated decision support and anomaly detection.
- Building a **real-time sensor fusion pipeline** for autonomous drone swarm navigation in disaster response scenarios.

## 🔬 Technologies Used


- **Frameworks**: ROS 2 (Robot Operating System), Multi-drone control suites.
- **AI / NLP**: LLMs (for situational analysis), Computer Vision (Object detection & tracking).
- **Core AI**: Perception and planning algorithms.
- **Languages**: Python, C++.
- **Domain**: Robotics, Disaster Response, Aerial Surveillance, Drone Swarms.
