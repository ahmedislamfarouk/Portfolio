# 🚀 The Ultimate AI Portfolio Architect Prompt

This prompt is engineered using the **UI-UX-PRO-MAX** framework to create a world-class, premium interactive portfolio for an AI Researcher and Professional Athlete.

---

## 🛠️ The Prompt

> **Task**: Build a "State-of-the-Art" Interactive Portfolio Website.
> **Role**: Senior Creative Web Engineer & AI UI/UX Specialist.
> **Design Theme**: "Liquid Glass & Obsidian Gold" (Premium Dark Mode).
> 
> **Core Identity**:
> - **User**: Ahmed Badr, AI Researcher & Professional Taekwondo Athlete.
> - **Focus**: Computer Vision, NLP, Robotics, Sensor Fusion.
> - **Special Highlight**: Medal of Excellence from the Egyptian President, African Games 1st place.
> 
> **Technology Stack (Required)**:
> - **Engine**: React + Vite (TypeScript).
> - **Styling**: Tailwind CSS v4 (Modern performance).
> - **Animations**: Framer Motion (Liquid morphing, staggered entry, hover-float).
> - **Icons**: Lucide-React (No emojis).
> 
> **Design Rules (UI-UX-PRO-MAX Standards)**:
> 1. **Color Palette**: 
>    - Background: `#0C0A09` (Deep Stone/Obsidian).
>    - Panels: `rgba(28, 25, 23, 0.7)` (Glassmorphism).
>    - Accent: `#CA8A04` (Premium Amber Gold for highlights).
>    - Text: `#FAFAF9` (Main) / `#A8A29E` (Muted).
> 2. **Typography**: 
>    - Headings: **Archivo** (Bold, tracking-tighter).
>    - Body: **Space Grotesk** (Modern, spaced-out).
> 3. **Visual Effects**: 
>    - "Liquid Glass": Backdrop blurs (20px+), 1px translucent borders (`white/10`).
>    - "Morphing Blobs": Large, slow-moving iridescent SVG blobs in background (Z-index -10).
>    - "Hover Synergy": Interactive project cards that lift and change border color to Gold.
> 
> **Functional Sections**:
> 1. **Hero**: Title: "Crafting Intelligence". Role: "AI Researcher". Needs an "Available for collaboration" badge.
> 2. **Technical Portfolio**: Filterable project grid (Robotics, NLP, Medical). Categorize projects (Renal Classification, Golf Cart, SkyVision Swarms).
> 3. **Research Frontiers**: Display papers "In Preparation" with status badges. Use Microscope icons.
> 4. **Milestones & Distinction**: Merge University and Professional Athletics. Highlight the 43 medals and Presidential Award in a Gold-themed card.
> 
> **Safety/Privacy constraints (CRITICAL)**:
> - **NO GPA MENTIONED**.
> - **NO CV DOWNLOAD LINK**. (Maintain professional mystery/privacy).
> - Clean, professional tone. No emojis as UI icons.
> 
> **Deliverable**: A fully functional `App.tsx` and `index.css` using the above directives. Ensure maximum aesthetic wow-factor.

### 💡 Why this prompt works:
- **Design Intelligence**: It specifies exact hex codes and font pairings suggested by the `ui-ux-pro-max` skill.
- **Constraints**: It explicitly bans emojis as icons and simple "plain" colors.
- **Interactivity**: It demands Framer Motion for that "premium" feel.
- **Context**: It provides the specific "Ahmed Badr" branding.
